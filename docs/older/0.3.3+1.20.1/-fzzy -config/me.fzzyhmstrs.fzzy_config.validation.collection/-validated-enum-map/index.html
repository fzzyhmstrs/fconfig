open class ValidatedEnumMap<K : Enum<*>, V>(defaultValue: Map<K, V>, keyHandler: Entry<K, *>, valueHandler: Entry<V, *>) : ValidatedField<Map<K, V>> , Map<K, V>

A Validated Map with Enum keys

NOTE: The provided map does not need to be an EnumMap, but it can be

Author

fzzyhmstrs

Since

0.2.0

Parameters

K

key type, any Enum

V

any non-null type with a valid Entry for handling

defaultValue

the default map

keyHandler

the Enum handler, typically a ValidatedEnum

valueHandler

the value handler, an Entry

See also

ValidatedEnumMap.Builder

using the builder is recommended
